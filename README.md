unRAID created packages

### Install

```bash
wget https://xxxxxxxx.txz
upgradepkg --install-new xxxx.txz
```

### Uninstall

```bash
removepkg xxxx.txz
```


### Install on unRAID boot

Place the .txz package into the /boot/extra folder. Create the folder if it didn't exists.
